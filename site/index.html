Use the library

Include one header and sort a vector.

Header-only algorithms return sorted vectors without mutating the original input.

#include "algorithm_sorting/sorting.hpp"

std::vector<int> values { 5, 1, 4, 2 };
auto sorted = algorithm_sorting::quick_sort(values);

Trace-ready

Algorithms can emit compare, swap, write, and done events through typed trace objects. The same trace data powers the WebAssembly visualizer above.

Included sorts

Bubble, selection, insertion, quick, counting, and radix sort are available as individual headers or through the sorting module.
