About

What this project demonstrates

The library exposes reusable C++ sorting algorithms, optional tracing, and small utility headers. This page compiles the C++ code to WebAssembly so the browser visualization is driven by the same implementation used by the library.

Algorithms

Available implementations

  • Bubble sort
  • Selection sort
  • Insertion sort
  • Quick sort
  • Counting sort
  • Radix sort

Architecture

Library shape

The project is organized as a header-only C++ library with focused public headers, a CMake test target, and a thin WebAssembly adapter for portfolio presentation.
